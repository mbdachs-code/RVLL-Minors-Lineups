# Changelog

## v1.0.9

- Tightened iPad print layouts for Safari and Chrome so the coach, dugout, and positions sheets are less likely to spill onto extra pages.

- Added a second defensive-position entry mode so coaches can build innings by position as well as by player name.
- Enlarged the positions-by-batting-order print sheet so it fills more of the page and is easier for kids to read.

- Changed the dugout batting-order print page to use a single lineup column on the left and the role reminders on the right.

- Added cache-busting for hosted assets so GitHub Pages updates show up more reliably after each release.

## v1.0.4

- Forced a fresh GitHub Pages redeploy to refresh the published site version.

## v1.0.3

- Added visible app versioning for local and printed comparisons.
- Refined browser batting-order editing and simplified roster editing.
- Improved print layouts for coach sheets, dugout sheets, and positions sheets.
- Updated team/home-visitor branding and print metadata behavior.
