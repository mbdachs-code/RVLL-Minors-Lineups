# Update Guide

Use this any time you want to update the lineup generator on GitHub.

```bash
cd "/Users/michaeldachs/Documents/Lineup Creator"
git add .
git commit -m "your update message"
git push
```

Example:

```bash
cd "/Users/michaeldachs/Documents/Lineup Creator"
git add .
git commit -m "Adjust coach print layout"
git push
```

Helpful check:

```bash
git status
```

If Git says `nothing to commit`, that means there are no new local changes to commit.
